# Informe de instalación del compilador y pruebas

## 1. Compilador seleccionado:
**GCC (GNU Compiler Collection)**

## 2. Versión del compilador:
**14.2.0**

## 3. Versiones del lenguaje C que soporta:
El compilador GCC soporta las siguientes versiones del lenguaje C:
- **C89 / C90**
- **C99**
- **C11**
- **C17**
- **C23** (por defecto en GCC 14)
